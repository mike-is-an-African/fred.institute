In a recipe, flour to sugar is 3:1. You use 150g flour. How much sugar?
3:1 → 1 part = 150÷3=50g. Sugar = 1×50 = 50g.

Finding One Quantity from a Ratio
When One Part is Known If you know one actual value, use the ratio to find others.

Ratio A:B = 3:5. If A = 18, find B.
1 unit = 18÷3 = 6
B = 5×6 = 30

A:B = 4:7. A = 24. Find B.
1 unit = 24÷4=6. B = 7×6 = 42.

Three-Part Ratios
Sharing in a 3-Part Ratio Same method — just add three parts.

Share $240 in ratio 1:2:3:
Total parts = 1+2+3 = 6
1 part = 240÷6 = $40
Shares: $40, $80, $120
Check: 40+80+120 = 240 ✓

Best Value Problems
Finding Best Value Calculate the UNIT rate (price per 1 item or per 1 kg) for each option. Lowest unit price = best value.

Small: $2.40 for 400g → $2.40÷400 = $0.006/g
Large: $3.50 for 700g → $3.50÷700 = $0.005/g
→ Large is better value (cheaper per gram)
3 pens for $1.80 or 5 pens for $2.75. Which is better value?

Key Points
  • Known one part: 1 unit = known value ÷ its ratio number
  • Three-part ratios: add all three, same process
  • Each part as fraction: part number ÷ total parts
  • Best value: calculate unit price (÷ by quantity), compare
  • Always check: all shares must add back to the total

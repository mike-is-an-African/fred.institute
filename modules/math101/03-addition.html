Addition — Key Vocabulary
Know the words before the maths.

Addition is the process of combining two or more quantities into a single total. It is one of the four basic arithmetic operations.

AddendThe numbers being added together. In 4 + 7 = 11, both 4 and 7 are addends.
SumThe result (answer) of an addition. In 4 + 7 = 11, the sum is 11.
Carrying (Regrouping)When digits in a column add up to 10 or more, you carry the tens digit into the next column to the left.
Addition is commutative: a + b = b + a. Order doesn't change the sum!
Addition is associative: (a + b) + c = a + (b + c). Grouping doesn't matter!
Real-world addition:
You buy a book for $12.50 and a pen for $3.75. Total = $12.50 + $3.75 = $16.25

Column Addition — Step by Step
The most reliable method for large numbers.
How to use Column Addition Step 1: Line up the digits by place value (units under units, tens under tens, etc.)
Step 2: Start from the RIGHTMOST column (units)
Step 3: Add the digits in that column
Step 4: If the sum ≥ 10, write the units digit and CARRY the tens digit to the next column
Step 5: Repeat for each column moving left
Example: 347 + 285
  3 4 7
+ 2 8 5
-------
  6 3 2
Units: 7+5=12 → write 2, carry 1
Tens: 4+8+1=13 → write 3, carry 1
Hundreds: 3+2+1=6 → write 6
Answer: 632

Adding Decimals & Negatives
The same column method — just align the decimal points!
Adding Decimals Rule: align the decimal points in a column. Fill gaps with zeros.

Example: 12.5 + 7.38
  12.50
+  7.38
------
  19.88
Adding Negative Numbers Adding a negative = subtracting: 5 + (−3) = 5 − 3 = 2
Two negatives together: (−4) + (−6) = −10
Think of a number line — negatives move LEFT
Always line up decimal points before adding. Add trailing zeros if needed so all numbers have the same decimal places.

Mental Addition Strategies
Smart shortcuts that make mental maths faster.
1
Partitioning
Break numbers into parts: 46+37 = (40+30)+(6+7) = 70+13 = 83
2
Rounding and Adjusting
Round one number up, then subtract: 47+38 = 47+40−2 = 87−2 = 85
3
Making 10s
Find pairs that make 10: 7+3+8+2+5 = (7+3)+(8+2)+5 = 10+10+5 = 25
4
Near doubles
6+7 = 6+6+1 = 12+1 = 13. Use doubles as a starting point.

Key Points to Remember
  • Sum = the result of addition. Addend = numbers being added.
  • Column addition: line up by place value, start from the right, carry when ≥ 10
  • Decimals: align the decimal points before adding
  • Adding a negative = subtraction: 5 + (−3) = 2
  • Commutative: a+b = b+a. Associative: (a+b)+c = a+(b+c)
  • Mental strategies: partitioning, rounding & adjusting, making 10s

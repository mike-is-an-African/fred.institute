What is a Variable?
A letter that represents an unknown or changing quantity.

Algebra uses letters (called variables) to represent numbers we don't know yet — or numbers that can change. Instead of solving one specific problem, algebra solves an entire category of problems at once.

VariableA letter (usually x, y, n, a, b…) that stands in for an unknown or changing number.

In "the price + tax = total", if price=x and tax=5: x + 5 = total
ConstantA fixed, known number that doesn't change. In 3x + 7, the constant is 7.
CoefficientThe number multiplying a variable. In 3x, the coefficient of x is 3. In −5y², the coefficient of y² is −5.
A single letter like x means 1×x. We write x, not 1x.

Substitution
Replacing variables with numbers to find the value.
What is Substitution? Replace each variable with a given number value, then calculate.

If x = 3 and y = 4, find 2x + 3y:
= 2(3) + 3(4)
= 6 + 12
= 18
Always use BODMAS when substituting: do powers first, then × and ÷, then + and −.
1
Find 5a − b when a=4, b=7
5(4)−7 = 20−7 = 13
2
Find x² + 2x when x=−3
(−3)²+2(−3) = 9+(−6) = 3
If p = 5 and q = 2, what is 3p − q²?

Like Terms & Collecting
Only add or subtract terms that are the same kind.
Like TermsTerms with exactly the same variable(s) AND power(s).
3x and 7x are like (both x).
3x and 3x² are NOT like (different powers).
4xy and 5xy are like (both xy).
Collecting Like TermsAdd or subtract the coefficients of like terms. Keep different terms separate.

5x + 3y − 2x + 7y
= (5x−2x) + (3y+7y)
= 3x + 10y
You CANNOT add x + x² — they are NOT like terms. They must stay separate.
Simplify: 4a + 3b − a + 5b

Multiplying & Dividing with Variables
Multiplying: multiply coefficients, then add powers of same variables.
3x × 4x = 12x² (3×4=12, x×x=x²)
2a × 5b = 10ab
2x² × 3x = 6x³
Dividing: divide coefficients, then subtract powers.
12x⁴ ÷ 3x² = 4x² (12÷3=4, x⁴÷x²=x²)
8ab ÷ 4b = 2a
What is 5x² × 3x?

Key Points
  • Variable: a letter representing an unknown number (x, y, n…)
  • Coefficient: the number multiplying a variable (in 3x, coefficient=3)
  • Constant: a fixed number (in 3x+7, constant=7)
  • Substitution: replace variables with numbers, use BODMAS
  • Like terms: same variable AND same power — only these can be added/subtracted
  • Multiply variables: multiply coefficients, ADD powers. Divide: subtract powers.

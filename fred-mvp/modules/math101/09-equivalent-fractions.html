C2 · Equivalent Fractions
What Are Equivalent Fractions?
Same value — different appearance.

Equivalent fractions are different fractions that represent the SAME amount. Imagine cutting a pizza into 2 slices and taking 1 (½), versus cutting into 4 slices and taking 2 (²⁄₄) — you have the same amount!

The Golden RuleMultiply OR divide BOTH the numerator AND denominator by the SAME non-zero number — the value NEVER changes.

½ = 2/4 = 3/6 = 4/8 = 5/10 = 50/100 — all identical!
This works because multiplying top AND bottom by the same number is like multiplying by 1 (e.g. ²⁄₂ = 1). Multiplying by 1 doesn't change the value.

Key Points
  • Equivalent fractions: same value, different appearance (½ = 2/4 = 3/6…)
  • Golden rule: multiply or divide BOTH top and bottom by the same number
  • To find equivalent: find what to multiply denominator by, do same to numerator
  • To compare: convert to same denominator, then compare numerators
  • To simplify: divide both by HCF
  • Fractions on number line: equivalent fractions land on same point

What is Perimeter?
The total distance around the outside of a shape.
PerimeterThe total length of all the sides of a shape. It is a LENGTH measurement — always in units like cm, m, km.
Perimeter Formulas Square: P = 4s
Rectangle: P = 2(l + w)
Triangle: P = a + b + c
Regular polygon with n sides: P = n × s
Circle (circumference): C = 2πr = πd
For any shape: add ALL the side lengths. If the shape is irregular, add every side individually.
Perimeter is 1D (a length). Area is 2D. Don't mix them up!

Circle Circumference
Circumference of a Circle C = 2πr (r = radius)
C = πd (d = diameter = 2r)
π ≈ 3.14159 (use 3.14 unless told otherwise)

If r = 5 cm: C = 2×π×5 = 31.4 cm
If d = 12 cm: C = π×12 = 37.7 cm

Perimeter of Compound Shapes
Compound (Composite) Shapes Shapes made from two or more basic shapes joined together.

To find perimeter: trace around the OUTSIDE boundary only. Add only the external sides — NOT shared internal sides.
Example: An L-shape made of two rectangles. Only trace the outer edges. Interior boundaries don't count.

Arc Length & Sector Perimeter
Arc Length Part of a circle's circumference.
Arc length = (θ/360) × 2πr
where θ = the angle at the centre in degrees

For a quarter circle (θ=90°): arc = (90/360)×2πr = ½πr
Sector Perimeter A "pizza slice" shape.
Perimeter = arc length + 2r

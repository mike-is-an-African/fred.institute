C6 · Data
Types of Data
Qualitative (Categorical) DataData described in words — not numbers.
Examples: colour, gender, favourite subject, nationality
Quantitative (Numerical) DataData expressed as numbers.

Discrete: only specific values possible (counted). E.g. number of children (0,1,2,3…)
Continuous: any value in a range (measured). E.g. height, weight, time
Primary vs Secondary DataPrimary: data you collect yourself (surveys, experiments)
Secondary: data already collected by others (government stats, databases)
The type of data determines which diagram to use. Discrete → bar charts. Continuous → histograms.

Frequency Tables
Frequency TableOrganises raw data by listing each category/value and its frequency (how many times it appears).

Tally marks: |||| = 4, ⁴⁄₅ = 5 (four with diagonal through)
Example — Favourite colour survey:
ColourFrequency
Red8
Blue14
Green6
Total28
In the table above, what fraction chose blue?
C6 · Data
Grouped Frequency Tables
Grouped Frequency TableUsed when data has a wide range. Data is grouped into class intervals.

Important: class intervals must not overlap and must cover all values.
Use inequality notation: 10 ≤ x < 20 (includes 10, excludes 20)
Example — Test scores (out of 50):
Score (x)Frequency
0 ≤ x < 103
10 ≤ x < 208
20 ≤ x < 3012
30 ≤ x < 407
In the table above, how many students scored 20 or more?

Pie Charts
Drawing a Pie Chart Total = 360° (the whole circle).
Angle for each sector = (frequency ÷ total) × 360°

Example: 40 students. 10 chose science.
Angle = (10÷40)×360° = 90°
1
Find total frequency
Add all frequencies together
2
Calculate each angle
(freq ÷ total) × 360°
3
Check: angles sum to 360°
All sectors must cover the full circle

Key Points
  • Qualitative=words. Quantitative=numbers (discrete=counted, continuous=measured).
  • Frequency table: each item and how many times it appears
  • Grouped frequency: class intervals, no overlap, cover all values
  • Pie chart angle = (freq ÷ total) × 360°
  • Modal class = class with highest frequency
